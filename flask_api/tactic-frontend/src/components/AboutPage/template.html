Tactic Extras*

* New name needed

This application holds several views, forms, and functions that are not otherwise possible within Tactic. It has views to add in Orders, search IMDb for new Titles, reorganize priorities, and more.

The reason this application is necessary in the first place is due to the restricting nature of the Tactic framework. Tactic is, in programming time, ancient. While some of its features and functionalities still hold up today, its front-end architecture is very outdated.

What this app aims to acheive is an easier, more responsive front end that utilizes Tactic on its back-end. The app is build with newer, fresher web development technology and methodology. This allows features like drag-and-drop, dynamic forms, and real-time updating which would otherwise be impossible in vanilla Tactic.

Because this application is using Tactic as its back-end (that is, the database it still running through Tactic, and this app merely fetches and pushes data to it), you can still use Tactic as you always have. The data is exactly the same. Note that this is not meant as a complete replacement of Tactic; not all Tactic's features will be migrated here. This is only meant to give access to features that Tactic would not normally allow.
